# Crowd Density Monitoring using AI

An AI-powered system to monitor and analyze crowd density in real-time.

## Features
- Real-time crowd detection and tracking
- Density analysis and visualization
- Web-based interface

## Installation

1. Clone this repository
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python realtime_camera.py
```

## Requirements
- Python 3.8+
- OpenCV
- PyTorch
- See requirements.txt for full list
